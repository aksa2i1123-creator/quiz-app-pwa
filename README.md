# Easy Math Quiz PWA

A simple progressive web app that lets users practice easy arithmetic questions. It includes offline caching via a service worker and a web app manifest for installability.

## How to run

1. Serve the folder from a local web server.
2. Open `http://localhost:PORT` in the browser.
3. Add to home screen when prompted.

## Files

- `index.html` — app structure and manifest link
- `styles.css` — visual styling
- `app.js` — quiz logic and PWA install handling
- `manifest.json` — install metadata
- `sw.js` — offline caching service worker
- `icon.svg` — app icon
